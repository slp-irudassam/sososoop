## 1. 개요

### 배경
현재 네이버 카페로 운영 중인 소소숲 커뮤니티를 독립 웹사이트로 이전한다. 자체 도메인에서 강의 판매·자료 배포·무료특강 신청을 한 곳에서 처리할 수 있도록 한다.

### 목표
- 강의 판매 채널 확보 (계좌이체 기반)
- 무료 자료 배포 (로그인 없이 다운로드)
- 무료특강 신청 접수
- 소소숲 브랜드 온라인 거점 구축

### 범위 (v1)
로그인·장바구니·자동결제 없음. 정적 웹앱에 가까운 구조로, 운영자가 직접 코드를 수정해 콘텐츠를 관리한다.

## 3. 페이지 구성

### 3-1. 홈 (`/`)
| 섹션 | 내용 |
|---|---|
| 히어로 슬라이더 | 3장 자동 전환 (소개 / 무료특강 / 자료실) |
| 퀵 메뉴 | 아이콘 6개: 자료실·강의신청·무료특강·소개·카카오채널·인스타그램 |
| 자료실 캐러셀 | 무료 다운로드 자료 카드 (더보기 링크) |
| 마퀴 배너 | 브랜드 슬로건 흐르는 텍스트 |
| 강의 캐러셀 | 유료 강의 카드 (더보기 링크) |
| 무료특강 CTA | 정규 강의 홍보용 무료특강 신청 유도 섹션 |

### 3-2. 소소숲 소개 (`/about`)
- 브랜드 소개, 운영자 소개(이루다쌤), 미션, 핵심 가치(성장·기록·나눔·실천·따뜻함)

### 3-3. 자료실 (`/resources`)
- 무료 다운로드 자료 그리드
- 로그인 없이 PDF 직접 다운로드
- 카테고리 뱃지: AI 활용·평가·기록·학습·임상

### 3-4. 강의 목록 (`/lectures`)
- 유료 강의 카드 그리드
- 각 카드: 카테고리·제목·설명·가격·수강시간

### 3-5. 강의 상세 (`/lectures/[id]`)
- 강의 소개
- 신청 방법 안내 (계좌이체 → 노션폼 → 카카오채널 '신청완료')
- 입금 계좌 표시
- 노션폼 링크 버튼
- 카카오채널 문의 버튼

### 3-6. 무료특강 신청 (`/free-lecture`)
- 정규 강의 홍보용 무료특강 (비정기 운영)
- 노션폼 링크 버튼
- 신청 절차 안내: 폼 작성 → 카카오채널 '신청완료' → 안내 수신

## 4. 핵심 기능

### 강의 판매 플로우
```
강의 상세 페이지
  → 노션 신청서 작성
  → 계좌이체 (카카오뱅크)
  → 카카오채널에 '신청완료' 입력
  → 운영자가 강의 안내 발송
```
> 별도 결제 모듈 없음. 수동 확인 후 발송.

### 무료 자료 다운로드
```
자료실 페이지 → 다운로드 버튼 클릭 → 파일 직접 다운로드
```
> 로그인·회원가입 불필요.

### 무료특강 신청 플로우
```
무료특강 페이지
  → 노션 신청서 작성
  → 카카오채널에 '신청완료' 입력
  → 운영자가 특강 안내 발송
```

## 10. 콘텐츠 관리 방법

코드 수정은 두 파일만 건드리면 됨:

- **강의 추가/수정**: `src/data/lectures.ts`
- **자료 추가/수정**: `src/data/resources.ts`

실제 PDF 파일은 `public/files/` 폴더에 업로드.
