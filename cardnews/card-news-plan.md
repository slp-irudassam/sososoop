# 카드뉴스 기획안 — P-FA Ⅱ 스터디 후기 (회고)

## 1. 목적
- **타겟**: 언어재활사 (현직·준비생), 특수교사
- **왜**: PFA 스터디를 마친 후 회고 + 나눔 자료 공유 + 7월 KOLRA 스터디 신청 유도
- **어떤 카드뉴스**: 강의 요약 아님. 이루다쌤이 스터디를 운영하면서 느낀 것, 준비한 것들을 나누는 회고 카드

## 2. 톤
- 따뜻하고 진솔한 후기 느낌
- 과하게 광고스럽지 않게, 동료에게 이야기하듯

## 3. 카드 구성 (7장)

| # | 유형 | 내용 요약 |
|---|------|-----------|
| 01 | Cover | "P-FA Ⅱ 스터디, 잘 마쳤어요 🌿" |
| 02 | 회고 | 어떤 스터디였는지 — 참여자 구성, 분위기 한 줄 |
| 03 | 자료 나눔 | 스터디에서 나눈 자료들 목록 |
| 04 | Dark (웹앱) | 발화분석 연습 웹앱 직접 제작 + 공유했어요 |
| 05 | 소감 | 준비하면서 뿌듯했던 것 / 참여자 반응 |
| 06 | Dark (CTA) | 7월 KOLRA 스터디 안내 + 신청 유도 |
| 07 | 마무리 | 감사 인사 + 팔로우 유도 |
